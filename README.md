## Overview
**Dish Deluxe** is a mobile recipe application built using **React Native with Expo**. It allows users to discover, search, and explore a variety of meals from different cuisines around the world.

The application integrates with the **TheMealDB API** to fetch real-time data including recipes, categories, ingredients, and images.

---

## Objectives
- Implement API integration in a mobile application  
- Design a clean and interactive user interface  
- Apply React Native and Expo development concepts  
- Demonstrate mobile app functionality using real-time data  

---

## Features
- Search meals by name  
- Browse recipes by category  
- Add and manage favorites  
- View detailed recipe instructions  
- Display meal images  
- Pull-to-refresh functionality  
- Clean and responsive UI  

---

## Tech Stack
- React Native (Expo)  
- TypeScript  
- REST API  
- TheMealDB API  

---

## API Used
**TheMealDB API**  
A free API that provides:
- Meal recipes  
- Categories  
- Ingredients  
- Images  

🔗 https://www.themealdb.com/api.php

## Installation & Setup

1. Clone the repository
```bash
git clone https://github.com/erlyx-12345/API_Quest.git
cd api-quest
```

2. Install dependencies
```bash
npm install
```
3. Start the app
```bash
npx expo start
```
